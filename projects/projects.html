HANDFISH ADVENTURE

Group

Game Jam

Godot

Context


Game for a Game Jam

Theme was Abyss

Creation


Group project (3 people)

Developed during a weekend

The project is based on an amazing animal: the handfish. Yes it is real.

It is a platformer where, just like in real life, the handfish doesn't swim but walks and dashes.

Note: the handfish physically can swim, he just doesn't do it.

The specie is going extinct because of starfish eating the eggs, so in the game, the handfish tries to save his eggs from a starfish.
